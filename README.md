# Intern Feedback Sentiment Analysis

## Overview
A sentiment analysis tool that automatically classifies intern feedback as 
Positive, Neutral, or Negative using Python and NLTK's VADER sentiment 
analyzer — turning hundreds of manually-read comments into instant, 
categorized results.

## The Problem
Reading through hundreds of feedback comments manually to gauge overall 
sentiment is slow and inconsistent. This project automates that process, 
classifying every comment in seconds instead of hours.

## What It Does
- Processes intern feedback comments in bulk
- Classifies each comment as Positive, Neutral, or Negative using VADER
- Outputs sentiment breakdown for quick interpretation

## Tech Stack
Python · NLTK (VADER Sentiment Analyzer) · Pandas
